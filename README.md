# LEADME Flutter 講座

IT未経験者でも大丈夫！ 
この講座では、スマホアプリを作るためのプログラミング言語「Flutter」を基礎から学んで、実際に動くアプリを作れるようになります。


## 講座の流れ

1. Dart文法をマスター

   Dartプログラミング言語の学習

2. 天気予報アプリを開発

   Flutter環境構築・UI構築・API連携

3. SNSアプリを開発

   本格的なアプリの実装・バックエンド連携

6. ハッカソンに向けて

   ２日間でアプリを実装するのに必要なこと

7. ブラッシュアップ

   アプリのブラッシュアップ・現場レベルにアプリの完成度を上げる
